# Exam Hall Allocation System
> Automated exam seat assignment for university students.

## Features
- Role-based dashboards (Admin, Student, Supervisor)
- MVC architecture (vanilla JS)
- RTL Arabic interface

## Tech Stack
HTML · CSS (Tailwind) · Vanilla JavaScript

## How to run
Open `index.html` in any browser. No build step needed.

## Project structure
```
js/
├── models/       ← data layer
├── controllers/  ← business logic
└── views/        ← UI rendering
```
